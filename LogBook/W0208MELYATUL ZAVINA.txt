AI Winter adalah periode berkurangnya minat terhadap penelitian dan pengembangan AI. Hal ini pernah terjadi 2 kali tahun 74 dan 90-an.

Father of AI adalah John McCarthy, beliau membuat bahasa pemrograman simbolik bernama list. Bahasa pemrograman simbolik secara matematis sangat berguna.
Tes Turing digunakan untuk menentukan apakah suatu program memenuhi syarat sebagai kecerdasan buatan dengan mengintrogasinya bersama dengan rekan manusianya. Kriteria mesin turing: jika terdapat sebuah computer dan 2 manusia diletakkan pada tempat terpisah, salah satu manusia harus bisa menebak mana chat dari manusia maupun computer. Jika manusia tidak bisa membedakan mana chat dari manusia maupun kompuuter, maka turing test lolos. Sophia adalah AI yang sudah lolos turing test.

Interpreter : Java, python, matlab, html

Perbedaan Jupyter Notebook dan Spyder yaitu; Jupyter untuk menulis catatan serta melakukan analisis data dalam file yang sama, sedangkan spyder merupakan IDE untuk scientific programing.
Kita dapat menulis catatan dalam jupyter notebook dengan kode markdown. Saat menggunakan markdown, # digunakan untuk menandai heading, tapi saat menggunakan opsi code, # digunakan untuk comment

Data structures merupakan cara untuk mengatur dan menyimpan data. Tipe data variabel menentukan operasi mana yang dapat diterapkan padanya.
Tipe data primitive; struktur data dasar, blok bangunan untuk manipulasi data dan berisi nilai data yang murni dan sederhana. Tipe data primitive terdiri dari; numbers (integer, float, complex), string, dan Boolean.
Tipe data non-promitive; berasal dari primitive data, digunakan untuk menyimpan kumpulan nilai.

LIST
List is mutable; list memiliki berbagai metode yang tersedia yang dapat digunakan untuk memanipulasi list itu sendiri seperti append(), insert(), remove(), dsb.
-	Dapat didefinisikan menggunakan tanda kurung siku []
-	Bisa berubah
-	Implikasi dari iterasi memakan waktu dalam list
-	Kurang hemat memori
-	Kesalahan dan perubahan yang tidak terduga dapat dengan mudah terjadi dalam list

TUPLE
Tuple tidak mendukung modifikasi sehingga merupakan objek yang tidak dapat diubah
-	Dapat didefinisikan menggunakan tanda kurung ()
-	Tidak bisa berubah
-	Implikasi dari iterasi jauh lebih cepat
-	Lebih hemat memori
-	Kesalahan dan perubahan yang tidak terduga jarang terjadi dalam tuple

STRING
-	Kumpulan abjad, kata, atau karakter
-	Merupakan primitive data types tapi mendukuing banyak fungsi dari koleksi (list dan tuple)
-	Dibuat dengan melampirkan urutan karakter dalam sepasang tanda kutip
-	Tidak dapat diubah, powerful

DICTIONARY
-	Dideklarasi dengan symbol {}
-	Bisa berubah (mutable) yaitu kita dapat memodifikasi baik menambahkan, mengurangi atau mengubah elemen dictionary setelah pembuatan

SET
-	Bersifat iterable
-	Bisa berubah (mutable)
-	Kumpulan value yang dipisahkan dengan koma dan diapit dalam tanda kurung kurawal {}
-	Tidak berurutan, bisa berubah dan tidak mendukung pengulangan elemen
-	Terutama digunakan untuk operasi teori himpunan seperti serikat pekerja, persimpangan, dll
-	Set adalah tipe data yang bisa diubah. Frozenset tidak bisa diubah

TIPE CONVERSION:
-	Implicit type conversion
-	Explicit type conversion
Contoh : list(s) untuk mengubah s menjadi sebuah list
